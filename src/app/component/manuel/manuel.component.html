name

ecy - ensar can yamak

synopsis

ecy [OPTIONS]

description

he has a bachelor's degree in computer science from TOBB ETU.

since 2021, he has been working as full-stack developer.

options

The following options are available:

--cv

display curriculum vitae

--social

display the social media accounts

environment

The following environment variables affect the execution of ecy:

LANG

java(default), js

FRAMEWORK

spring(default), angular

examples

Show the related social media accounts:

$ ecy --social

exit status

The ecy command exits 0 on success, and > 0 if an error occurs.

see also

contact(1), cv(1), social(3)
